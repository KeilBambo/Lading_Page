One Page Site

Ideal para negócios de pequeno ou médio porte. O "one page site" é um site onde voce inclui todas informações que o visitante precisa de forma objectiva e criativa em uma única página

Wireframe

Wireframe é uma representação visual básica de uma página. Ele deve ser feito de maneira mais simples possível, mostrando apenas o essencial, como uma espécie de rascunho, sem cores ou imagens. Clique no botão ou scaneie o QR Code para obter o wireframe desta página.
